# VibeBoard Seed Submissions

A research-compiled list of 30 real AI-built ("vibe-coded") tools suitable for seeding VibeBoard. All entries were found through web search (Twitter/X, Product Hunt, Hacker News, Indie Hackers, curated GitHub lists) between roughly November 2025 and April 2026. Where a field could not be verified, it is marked "not found" rather than fabricated.

Primary sources consulted:
- `levz0r/awesome-vibecoded-apps` GitHub list
- `jam.pieter.com` (Levels' Vibe Jam)
- `walzr.com` (Riley Walz's portfolio)
- Product Hunt leaderboards for Feb/Mar/Apr 2026
- Hacker News "Show HN" front page (Dec 2025 / Jan 2026)
- Twitter/X threads referenced in each entry

Notes on verification:
- "Fully verified" = name + live URL + creator all confirmed.
- "Partial" = live URL confirmed but creator/handle "not found".
- Screenshot URLs point to the tweet or PH post that contained the embedded media (never a fabricated direct image URL).
